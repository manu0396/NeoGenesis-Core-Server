# Design History File (DHF) Index

## Controlled registry
- DHF artifacts are registered via `/regulatory/dhf`.
- Metadata captured:
  - `artifactType`
  - `artifactName`
  - `version`
  - `location`
  - `checksumSha256`
  - `approvedBy`
  - `approvedAt`

## Mandatory sections
- Requirements and traceability matrix.
- Architecture and cybersecurity controls.
- Verification/validation reports.
- Risk management and CAPA outcomes.
